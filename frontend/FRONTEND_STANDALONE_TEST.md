### 测试4: Toast通知系统 ✅

**步骤**:
1. 打开浏览器控制台
2. 执行以下代码测试Toast:

```javascript
// 测试成功通知
window.dispatchEvent(new CustomEvent('show-toast', {
  detail: { type: 'success', message: '操作成功！' }
}));

// 测试错误通知
window.dispatchEvent(new CustomEvent('show-toast', {
  detail: { type: 'error', message: '操作失败，请重试' }
}));

// 测试警告通知
window.dispatchEvent(new CustomEvent('show-toast', {
  detail: { type: 'warning', message: '请注意检查输入' }
}));

// 测试信息通知
window.dispatchEvent(new CustomEvent('show-toast', {
  detail: { type: 'info', message: '这是一条提示信息' }
}));
```

**预期结果**:
- ✅ Toast从右侧滑入
- ✅ 显示正确的颜色和图标
- ✅ 5秒后自动消失
- ✅ 可以手动关闭
- ✅ 多个Toast可以堆叠显示
