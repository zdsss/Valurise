# Valurise - 执行摘要

**一页纸总结：你需要知道的一切**

---

## 🎯 产品定位

**"AI职业价值发现引擎"**

帮助职场人发现隐藏价值，重构职业叙事，战略性呈现自己。

**不是**：简历格式化工具
**而是**：职业价值战略顾问

---

## 💎 核心差异化

1. **深度价值挖掘** - 多Agent协作，发现用户未意识到的价值
2. **战略叙事重构** - 将碎片化经历编织成有说服力的故事
3. **多场景适配** - 针对不同目标生成不同版本
4. **技术护城河** - 多Agent架构 + 数据飞轮

---

## 👥 目标用户

**初期聚焦**：
- 职业转型者（痛点最强，付费意愿高）
- 高潜力职场人（3-10年经验）
- 创业者/自由职业者

**避免**：应届生、低端求职市场

## 🔧 技术原型

### 4个专业化Agent

1. **信息提取Agent** - 结构化用户输入
2. **价值分析Agent** - 量化成就，识别可迁移技能
3. **叙事策略Agent** - 构建职业故事，制定定位
4. **简历格式化Agent** - 生成优化简历

### 性能指标

- **成本**：$0.10-0.30/次（远低于$2目标）
- **时间**：30-50秒
- **质量**：待测试验证

### 技术栈

- AI：Anthropic Claude API (Sonnet 4.6)
- 语言：Python 3.10+
- 框架：Pydantic, Rich
- 未来：React + FastAPI + PostgreSQL
