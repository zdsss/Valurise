## 🚀 立即开始（5分钟）

### 步骤1：安装依赖

```bash
cd prototype
pip install -r requirements.txt
```

### 步骤2：配置API Key

1. 复制环境变量模板：
```bash
cp .env.example .env
```

2. 编辑`.env`文件，填入你的Anthropic API key：
```
ANTHROPIC_API_KEY=sk-ant-xxxxx
```

获取API key：https://console.anthropic.com/

### 步骤3：运行原型

```bash
python main.py
```

你将看到：
- 🔍 Agent 1: 提取用户信息
- 💎 Agent 2: 分析职业价值
- 📖 Agent 3: 制定叙事策略
- 📄 Agent 4: 生成简历版本
- 💰 成本分析报告

结果保存在`output/valurise_output.json`

## 🧪 测试建议

### 测试场景

准备5个不同的测试案例：

1. **职业转型者**
   - 从技术转产品
   - 从传统行业转互联网

2. **高潜力职场人**
   - 3-5年经验
   - 寻求晋升或跳槽

3. **创业者**
   - 多元化背景
   - 需要个人品牌呈现

4. **应届生**（对比测试）
   - 经验较少
   - 看产品是否适用

5. **高管**（对比测试）
   - 丰富经验
   - 看输出质量

### 评估维度

对每个测试案例评分（1-5分）：

1. **信息提取准确性**
   - 是否正确提取了关键信息？
   - 是否识别了缺失信息？

2. **价值分析深度**
   - 是否发现了隐藏价值？
   - 量化是否合理？
   - 可迁移技能是否准确？

3. **叙事策略质量**
   - 职业故事是否有说服力？
   - 定位是否清晰？
   - 差异化是否明显？

4. **简历优化效果**
   - 内容是否优于原始输入？
   - ATS关键词是否合理？
   - 格式是否专业？

5. **整体价值感知**
   - 用户会为此付$99吗？
   - 相比人工服务如何？
