关键数据
--------

产品定位：AI职业价值发现引擎
目标用户：职业转型者、高潜力职场人
定价策略：$49（基础）/ $99（专业）/ $199（高级）

技术指标：
- 预期成本：$0.10-0.30/次
- 处理时间：30-50秒
- 目标质量：>3.5/5

商业目标（3个月）：
- 付费用户：100人
- 转化率：>15%
- 预期收入：~$9,400
- 净利润：~$6,350


下一步行动
----------

今天：
1. cd prototype
2. pip install -r requirements.txt
3. cp .env.example .env
4. 编辑.env，填入API key
5. python main.py

本周：
1. 测试5个不同场景
2. 邀请5-10个朋友测试
3. 收集反馈和付费意愿
4. 决定是否继续（Go/No-Go）
